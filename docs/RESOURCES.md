## 1. 植物精灵

### 概述

- **动态着色**：每种花只制作一套"白底/灰底"精灵，运行时通过 `modulate` 属性动态着色，无需为每种颜色制作独立精灵
- **生长阶段**：每种花 5 个阶段（种子 / 发芽 / 幼苗 / 成株 / 开花）
- **形状差异**：代码中 `shape` 参数 (0-3) 决定形状，用同一套精灵的不同帧表示

### 精灵规格

| 用途 | 推荐尺寸 | 格式 | 备注 |
|------|---------|------|------|
| 花圃内显示 | 64×64 px | PNG 带透明通道 | 开花阶段可适当放大 |
| 桌面花瓶显示 | 48×64 px | PNG 带透明通道 | 竖向稍高 |
| 图鉴缩略图 | 32×32 px | PNG 带透明通道 | 可复用花圃精灵缩放 |
| 仓库/选花列表 | 32×32 px | PNG 带透明通道 | 紧凑排列 |

### 精灵风格要求

- **卡通圆润风格**，描边清晰
- **白底/浅灰底**，方便动态着色
- **5 个生长阶段风格统一**，种子最小最简单，开花最精细
- **shape 形状差异明显**：尖瓣/圆瓣/锯齿/异形一眼可辨
- **size 大小差异明显**：小=紧凑，中=普通，大=饱满

### 图鉴剪影

- 使用开花阶段精灵，灰度化 + 阈值处理 → 纯黑轮廓
- 共 27 张（与开花阶段精灵一一对应）

## 临时替代方案

在正式美术资源就位前，使用以下替代方案：

| 正式资源 | 临时替代 |
|---------|---------|
| 植物精灵 | Emoji 文字（🌸🌷🌺🌻等）+ Label 节点 |
| 花圃背景 | 纯色 + GridMap 绘制 |
| 浇水特效 | 纯色圆点缩放动画（Tween） |
| 按钮图标 | 文字按钮（"收入仓库"） |
| 音效 | 无声（SFXPlayer 占位已埋好触发点） |

> **当前状态**：游戏已使用 Emoji 临时素材运行，P0 正式资源优先替换植物精灵和场景背景。
