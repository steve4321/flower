# Flower Desktop - 新手入门指南

## 1. 游戏简介
Flower Desktop 是一款零压力的桌面养花游戏。种花浇水，培育新品种，在桌面花瓶里插花欣赏。

### 核心特色
- 💚 零压力 — 花不会死、不会枯萎，不玩也不会有任何损失
- 🌱 简单操作 — 只有浇水和培育两个核心操作
- 🎲 培育惊喜 — 每次培育结果都不同，有概率出稀有花
- 💐 桌面花瓶 — 在花瓶里插花装饰桌面，花会轻轻摇曳
- 📖 收集图鉴 — 27种花等你去发现

### 三个空间
- **花圃** — 主游戏界面，种花、浇水、管理
- **培育室** — 从仓库选花培育，获得新种子
- **桌面** — 花瓶插花展示

## 2. 基础操作

### 2.1 浇水
点一下正在生长的花就是浇水。从种子到开花需要10次浇水：

| 阶段 | 浇水次数 | 视觉 |
|-----|---------|------|
| 种子 | 2 | 🟤 |
| 发芽 | 2 | 🌱 |
| 幼苗 | 3 | 🌿 |
| 成株 | 3 | 🪴 |
| 开花 | 永久 | 🌸 |

### 2.2 管理开花的花
点击开花的花会出现操作菜单：
- **📦 收入仓库** — 把花存到花仓库，可以在培育室使用
- **🗑 移除** — 移除这株花

### 2.3 右键操作
右键点击花圃里的花可以直接移除。

## 3. 花仓库

开花后的花可以收入花仓库。仓库没有容量限制，存在里面的花可以：
- 在培育室用来培育新花
- 随时取出放回花圃

## 4. 培育系统

### 4.1 培育流程
1. 进入培育室（花圃底部按钮）
2. 从仓库选两朵花作为亲本
3. 点击"培育"按钮
4. 直接获得新种子，加入种子库
5. 花不消耗，可以反复培育

### 4.2 培育规则
- 同属同种才能杂交出特殊品种
- 花和多肉不能杂交
- 培育结果有三层：
  - 70% 混色（同品种不同颜色）
  - 27% 同组杂交（新品种）
  - 3% 稀有变异

### 4.3 培育组
| 培育组 | 包含品种 |
|--------|---------|
| 蔷薇系 | 玫瑰、樱花、牡丹 |
| 百合系 | 郁金香、百合、风信子 |
| 菊系 | 雏菊、向日葵、康乃馨、格桑花、满天星 |
| 兰系 | 薰衣草、蝴蝶兰 |
| 多肉系 | 观音莲、玉露、熊童子、玉龙观音 |
| 仙人掌 | 仙人掌 |

## 5. 桌面花瓶

### 5.1 插花
- 点击桌面花瓶 → 弹出插花选择
- 列出花圃里所有开花的植物
- 可以添加多朵花到花瓶
- 花不消耗，还在花圃里
- 每朵花有独特的摇曳动画

### 5.2 换花
再次点击花瓶可以继续添加或更换花朵。

## 6. 图鉴

### 6.1 收集
每次开花如果是新品种，自动收录图鉴。点击花圃底部的"📖 图鉴"按钮查看。

### 6.2 图鉴界面
- 已发现的花正常显示彩色和名称
- 未发现的显示为剪影
- 可按类别筛选：全部/花卉/多肉/稀有

## 7. 花圃扩展
- 初始6格
- 每收集5种新花，自动扩展+3格
- 最大可扩展到20格

## 8. 稀有花
| 名称 | 触发条件 |
|------|---------|
| 彩虹玫瑰 | 蔷薇系培育，RGB均>200 |
| 暗夜曼陀罗 | 任意培育，RGB均<50 |
| 金色向日葵 | 菊系培育，R>200 G>180 B<50 |
| 月光百合 | 百合系培育，R<80 G>180 B>180 |
| 永恒之花 | 3%概率的fallback |

稀有花不能被攻略，只能被遇见！

## 9. FAQ
Q: 花会死吗？ A: 不会！零压力设计，花永远不会死。

Q: 不玩会怎样？ A: 什么都不会发生，进度永久保存。

Q: 培育后花会消失吗？ A: 不会，培育不消耗花朵。

Q: 如何获得新种子？ A: 通过培育室培育，结果直接变种子。

Q: 存档在哪？ A: 每次操作自动存档，无需手动保存。
