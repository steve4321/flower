# Flower Desktop - 桌面养花游戏

## 1. 项目概述

**类型**: 桌面悬浮插件式养花游戏
**引擎**: Godot 4.x
**平台**: Windows/Linux
**显示**: 花盆形状悬浮窗，桌面底部显示

## 2. 核心设计

### 2.1 视觉风格
- 卡通风格
- 花盆形状悬浮窗
- 半透明背景
- 桌面底部置顶显示

### 2.2 植物类型
- **花卉**: 10-20种基础花
- **多肉/仙人掌**: 5-10种

### 2.3 培育系统
- **基因遗传系统**
  - 每株植物有基因属性（颜色、形状、大小、花期）
  - 杂交时基因随机组合
  - 可培育出稀有变种
- **稀有花概率**: 5%

### 2.4 花盆系统
- **花盆数量**: 3-5个
- 每个花盆独立种植
- 可同时培育不同植物

## 3. 玩法设计

### 3.1 日常操作（点击按钮）
- **浇水**: 每24小时需要浇水
- **施肥**: 促进生长和变异
- **晒太阳**: 积累能量
- **修剪**: 保持植物健康

### 3.2 培育流程
1. 选择两株植物放入培育槽
2. 等待培育时间
3. 获得新植株（可能产生变异）
4. 稀有变异概率：5%

### 3.3 生长周期
- 种子 → 发芽 → 幼苗 → 成株 → 开花
- 每个阶段需要不同照料
- 开花后可进行杂交

## 4. 基因系统

### 4.1 基因属性
| 属性 | 说明 | 变异范围 |
|-----|------|---------|
| 颜色 | R/G/B三原色 | 0-255 |
| 形状 | 花瓣形状 | 尖/圆/锯齿 |
| 大小 | 植物尺寸 | 小/中/大 |
| 花期 | 开花周期 | 短/中/长 |

### 4.2 遗传规则
- 显性基因优先
- 隐性基因可能隔代遗传
- 稀有基因需要特定组合触发

## 10. 数值设计

### 10.1 照料消耗
| 操作 | 效果 | 冷却 |
|-----|------|------|
| 浇水 | 水+30% | 即时 |
| 施肥 | 肥+20% | 12小时 |
| 晒太阳 | 阳+50% | 即时 |
| 修剪 | 恢复健康 | 24小时 |

### 10.2 生长参数
| 阶段 | 时长 | 所需照料 |
|-----|------|---------|
| 种子 | 1小时 | 水 |
| 发芽 | 2小时 | 水+阳 |
| 幼苗 | 4小时 | 水+肥 |
| 成株 | 8小时 | 水+肥+阳 |
| 开花 | 永久 | 维持 |

### 10.3 基因参数
- 颜色基因: RGB各8位 (0-255)
- 形状基因: 显性/隐性
- 大小基因: 数量性状
- 稀有触发: 特定基因组合
