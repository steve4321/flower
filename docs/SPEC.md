# AIMMORPG - 放置挂机RPG

## 1. 项目概述

**类型**: 放置挂机RPG，动态桌面插件风格
**引擎**: Godot 4.x
**平台**: Windows/Linux
**显示**: 悬浮插件，半透明，桌面底部显示

## 2. 核心设计

### 2.1 视觉风格
- 像素风格
- 半透明弧形底部悬浮插件
- 桌面底部置顶显示

### 2.2 战斗系统
- 战法牧铁三角，全AI自动战斗
- 横版刷怪：杂兵波次 → Boss
- 战斗速度：快速
- 失败重试当前Boss前杂兵

### 2.3 角色系统
- 职业：战士、法师、牧师
- 等级提升属性
- 装备驱动成长

### 2.4 装备系统
- 品质：白→绿→蓝→紫→橙
- 可为3个角色分别配置装备

### 2.5 主城系统
- 铁匠：强化装备
- 药剂师：购买药水
- 仓库：存放装备
- 副本入口

## 3. 副本设计

### 首发副本：死亡矿井
- Wave 1: 3只豺狼人
- Wave 2: 4只迪菲亚
- Wave 3: 5只矿工
- Wave 4: Boss - 范克里夫
- 通关奖励宝箱

## 6. 技术方案

### 6.1 窗口化显示
- Godot `--embedded` 模式
- 自定义窗口样式：无边框、透明背景
- 置顶显示 (always on top)
- 弧形底部边缘通过shapes绘制

## 7. 角色属性

### 战士 (Warrior)
- HP: 高
- MP: 低
- 定位: 坦克
- 技能: 嘲讽、顺劈、雷霆一击

### 法师 (Mage)
- HP: 低
- MP: 高
- 定位: 输出
- 技能: 火球术、奥术飞弹、冰霜新星

### 牧师 (Priest)
- HP: 中
- MP: 中
- 定位: 治疗
- 技能: 治疗、恢复、真言术：盾
