# Flower Desktop - 桌面养花游戏

> 桌面悬浮插件式基因培育养花游戏

## 1. 项目概述

### 1.1 基本信息
| 项目 | 内容 |
|-----|------|
| 项目名称 | Flower Desktop |
| 类型 | 桌面悬浮插件 / 放置养成 |
| 引擎 | Godot 4.2+ |
| 语言 | GDScript |
| 平台 | Windows / Linux |
| 显示模式 | 悬浮窗，桌面底部置顶 |

### 1.2 核心特色
- **花盆形态悬浮窗** - 模拟真实花盆形状，桌面底部显示
- **基因遗传系统** - 培育杂交，基因自由组合
- **稀有花挑战** - 5%概率培育稀有变种
- **轻量挂机** - 每日简单照料，休闲养花

### 1.3 设计理念
- 休闲第一：不强迫每日登录，没有惩罚
- 收集驱动：解锁全部花卉图鉴
- 惊喜元素：稀有花杂交带来惊喜和期待

## 4. 游戏系统

### 4.1 植物生长周期

```
种子 ──→ 发芽 ──→ 幼苗 ──→ 成株 ──→ 开花
  │       │       │       │       │
  │       │       │       │       └── 可收获种子
  │       │       │       │           可进行杂交
  │       │       │       │
  │       │       │       └── 需要完整照料
  │       │       │
  │       │       └── 需要水和阳光
  │       │
  │       └── 需要水分
  │
  └── 初始状态
```

| 阶段 | 时长 | 所需照料 | 可选操作 |
|-----|------|---------|---------|
| 种子 | 1小时 | 水分>20% | 加速(等待) |
| 发芽 | 2小时 | 水分>30%, 阳光>20% | 继续等待 |
| 幼苗 | 4小时 | 水分>40%, 肥料>10% | 继续等待 |
| 成株 | 8小时 | 全部>50% | 可修剪 |
| 开花 | 永久 | 维持>30% | 收获/杂交 |

### 4.2 照料系统

| 操作 | 效果 | 冷却 | 方式 |
|-----|------|------|------|
| 浇水 | 水分+30% | 即时 | 点击按钮 |
| 施肥 | 肥料+20% | 12小时 | 点击按钮 |
| 晒太阳 | 阳光+50% | 即时 | 点击按钮 |
| 修剪 | 清除枯叶，恢复健康 | 24小时 | 点击按钮 |

**照料状态计算**：
- 每小时自然消耗：水分-2%，肥料-1%，阳光-3%
- 状态影响生长速度：
  - 单项<20%：生长停滞
  - 单项<10%：健康下降
  - 全部>50%：生长加速10%

### 4.3 基因遗传系统

#### 4.3.1 基因类型
| 基因 | 描述 | 变异方式 |
|-----|------|---------|
| 颜色(R) | 红色基因 | 0-255 |
| 颜色(G) | 绿色基因 | 0-255 |
| 颜色(B) | 蓝色基因 | 0-255 |
| 形状 | 花瓣形状 | 显性/隐性 |
| 大小 | 植物尺寸 | 数量性状 |
| 花期 | 开花周期 | 短/中/长 |

#### 4.3.2 遗传规则
```
Parent A 基因: [Rr, Gg, Bb, 形状Aa, 大小]
Parent B 基因: [rr, GG, BB, 形状aa, 大小]

Child 基因:
- 每对等位基因随机继承一个
- 颜色: 50% 来自A，50% 来自B
- 形状: 显性基因优先
- 5% 概率发生突变
```

#### 4.3.3 稀有基因触发条件
| 稀有花 | 触发条件 | 概率 |
|-------|---------|------|
| 彩虹玫瑰 | R/G/B三色基因均>200 + 特定形状组合 | 5% |
| 暗夜曼陀罗 | R<50, G<50, B<50 + 夜间杂交 | 5% |
| 金色向日葵 | R>200, G>180, B<50 | 5% |
| 月光百合 | 银色基因表达(R<50, G>200, B>200) | 5% |
| 永恒之花 | 所有属性达到极值 | 5% |

### 4.4 收获与种子系统

- **收获条件**：植物开花后可收获
- **收获产物**：1-3颗种子（随机）
- **种子继承**：种子保留母株50%基因
- **种子变异**：收获时5%概率基因突变

### 4.5 商店系统（简化）

| 物品 | 价格 | 效果 |
|-----|------|------|
| 基础肥料 | 10金币 | 肥料+50% |
| 高级肥料 | 50金币 | 肥料+100% |
| 生长激素 | 30金币 | 加速生长1小时 |
| 稀有催化剂 | 200金币 | 稀有概率+2% |

**金币获取**：
- 收获普通花：5-15金币
- 收获稀有花：50-100金币
- 首次图鉴收录：20金币
